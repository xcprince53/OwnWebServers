# Cursor 计费指南：如何高效管理订阅与优化成本

## 计费常见问题解答

以下是关于 Cursor 订阅与计费管理的所有关键信息。

### 如何访问计费设置？

通过 Cursor 的仪表板，点击“计费”按钮即可进入计费门户。这里是一个安全的环境，供您处理所有与计费相关的任务。

![如何访问Cursor计费设置](https://bbtdd.com/img/390930874244550.webp)

### Cursor 的计费周期是什么？

Cursor 的计费周期可以是按月或按年，从您订阅之日起开始计算。对于拥有多个席位的商业账户，当团队规模发生变化时，我们采用按比例计费，确保您只需为每个席位在计费周期内的实际使用时间付费。

### 商业账户的团队席位如何管理？

商业账户采用按席位计费模式，每个团队成员需要一个席位许可。当在计费周期中途添加新成员时，您只需为他们在该周期内的剩余时间付费。团队管理员可以通过仪表板直接管理席位。

### 可以在月付和年付之间切换吗？

当然可以！以下是操作步骤：

**商业版：**
1. 前往设置页面。
2. 在账户部分，点击“高级”，然后选择“升级到年付”。

![Cursor商业版](https://bbtdd.com/img/0606371935.webp)

> 请注意，您只能自助从月付切换到年付。如需从年付切换到月付，请联系我们：hi@cursor.com。

### 试用期如何运作？

Cursor 的专业版试用期持续 14 天，或直到达到定价页面上详述的使用配额。试用期结束后，您需要升级才能继续使用专业版功能。您可以在试用期内随时升级。

### 如何查找我的发票？

所有的计费历史都可以在计费门户中查看，您可以随时查看和下载当前及过去的发票。

### 可以自动接收发票邮件吗？

目前，发票需要从计费门户手动下载。我们正在开发自动发票邮件功能，一旦上线，您将可以选择启用！

### 如何更新我的计费信息？

您可以通过计费门户更新支付方式、公司名称、地址和税务信息。我们使用 Stripe 作为支付处理商，确保交易安全。请注意，计费信息的更改仅影响未来发票，历史发票无法修改。

### 如何取消我的订阅？

您可以通过计费门户中的“取消订阅”按钮直接取消订阅，您的访问权限将持续到当前计费期结束。

### 遇到其他计费问题如何解决？

对于本文未涵盖的任何计费问题，请发送邮件至 hi@cursor.com。请附上您的账户详细信息和具体问题，我们的团队将尽快为您提供帮助！

### 我可以申请退款吗？

您可以通过计费门户中的“取消订阅”按钮自助申请退款。如果不在自助退款窗口期内，请通过邮件联系我们：hi@cursor.com，我们将协助您处理。
